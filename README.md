# Image Decoding Performance

## Installation

```sh
npm install
```

## Preparation

Create resized images in various formats:

```sh
npm run images
```

## Measure

Start server for index.html and image files:

```sh
npx serve
```

Run decoding measurement 5x for each image:

```sh
npm run measure
```
