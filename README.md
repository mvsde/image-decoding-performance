# Image Performance Measurement

## Requirements

* [Node.js](https://nodejs.org/)

## Installation

```sh
npm install
```

## Preparation

Create resized images in various formats:

```sh
npm run images
```

## Measure

Run performance measurement 5x for each image:

```sh
npm run measure
```
